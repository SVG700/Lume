# LUME Landing Page

Modern startup-style product landing page for the Design Thinking project **LUME**.

## Stack
- Next.js (App Router)
- Tailwind CSS
- TypeScript

## Run locally
1. Install dependencies:
   npm install
2. Start development server:
   npm run dev
3. Open:
   http://localhost:3000

## Build
- npm run build
- npm run start

## Important customization
Update these values in `app/page.tsx`:
- `googleFormUrl` for the external survey button
- `googleFormEmbedUrl` for the embedded survey form
